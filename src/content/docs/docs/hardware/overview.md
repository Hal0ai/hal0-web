hal0 targets three classes of home hardware in v1:

1. **AMD Strix Halo** — first-class, unified-memory iGPU (the crown jewel).
2. **AMD discrete GPUs** — ROCm or Vulkan.
3. **NVIDIA discrete GPUs** — CUDA or Vulkan.

See [Strix Halo](/docs/hardware/strix-halo/) for the canonical reference platform.
