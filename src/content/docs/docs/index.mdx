Welcome to the hal0 documentation. hal0 is a polished, reliable
inference platform for running LLMs at home — it manages model slots,
exposes an OpenAI-compatible API, and ships with a built-in dashboard
and prewired chat UI.

> **Status:** v0.1.0-alpha — shipping. The cosign-signed release pipeline
> is wired end-to-end; the install command actually installs. Expect rough
> edges: APIs may shift, some features (full FLM NPU coverage, ROCm/CUDA
> toolbox parity) are still landing. Pages marked "Coming soon" are stubs.
