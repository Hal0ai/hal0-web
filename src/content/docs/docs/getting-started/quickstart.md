The one-line installer is the supported v1 path:

```sh
curl -fsSL hal0.dev/install | bash
```

Once the installer completes, the dashboard is reachable at
`http://hal0.local:8080`, and the prewired OpenWebUI chat at
`http://hal0.local:3001`.
